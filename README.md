# Digital_Earth
3D Earth visualization built with WebGL and Three.js

## Features

- **Realistic Earth Rendering** - High-quality 3D globe with detailed textures
- **Dynamic Atmosphere** - Glowing atmospheric effects and cloud layers
- **Interactive Controls** - Rotate, zoom, and explore the planet
- **Real-time Effects** - Bloom lighting and ambient occlusion
- **Ecosystem Visualization** - Environmental data overlay
- **Performance Monitoring** - Built-in FPS counter and stats

## Quick Start

Simply open `app.html` in a modern web browser. No installation or build process required!

```bash
# Clone the repository
git clone https://github.com/Igriscodes/Digital_Earth.git

# Navigate to the directory
cd Digital_Earth

# Open in browser
open app.html
```

## Controls

- **Auto Rotate** - Toggle automatic planet rotation
- **Rotation Speed** - Adjust spin velocity
- **Bloom Intensity** - Control atmospheric glow
- **Ocean Level** - Modify sea level visualization
- **Cloud Speed** - Adjust cloud movement
- **Show/Hide Layers** - Toggle clouds, ecosystem, and stats

## Technologies

- Three.js - 3D graphics library
- WebGL - Hardware-accelerated rendering
- Pure JavaScript - No dependencies beyond Three.js

## Requirements

- Modern web browser with WebGL support
- No server or build tools needed
